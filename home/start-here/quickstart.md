# Quickstart

Use this page to validate the core authoring path in GitBook.

{% stepper %}
{% step %}
### Create a draft

Open a page, make a small update, and leave a clear summary of what changed.
{% endstep %}

{% step %}
### Request review

Ask one teammate to review the change request before it goes live.
{% endstep %}

{% step %}
### Publish

Merge the change request, publish the site, and check the public URL.
{% endstep %}

{% step %}
### Ask a question

Use Ask to confirm the page answers common visitor questions in plain language.
{% endstep %}
{% endstepper %}

## Good first edits

| Edit | Why it is useful |
| --- | --- |
| Add a short FAQ answer | Easy to verify in Ask. |
| Rename a page group | Confirms sidebar navigation behavior. |
| Add a card link | Checks landing-page layout and previews. |
