# Quality checklist

Use this before sharing the site.

| Area | Pass condition |
| --- | --- |
| Homepage | Has a clear brand, cover image, and useful cards. |
| Sidebar | Uses logical groups and readable labels. |
| Pages | Each page has a description and one clear intent. |
| Search | Common questions return useful answers. |
| Branding | Logo, favicon, colors, and links are consistent. |
| Access | The intended visitor can open the site. |
