# AI search and MCP

AI answers work best when pages are direct, specific, and organized around real questions.

## Ask-ready writing

| Do | Avoid |
| --- | --- |
| Write the answer in the first paragraph. | Hiding the answer under long setup text. |
| Use exact product terms consistently. | Switching between synonyms for the same thing. |
| Keep steps atomic and ordered. | Combining multiple actions in one step. |
| Link to the next relevant page. | Ending with no path forward. |

## Suggested questions

* How do I invite my team to a workspace?
* How do I publish a change safely?
* How do I configure the site branding?
* What should I check before sharing a link?
