# Branding and navigation

The Nimbus test site uses a polished blue and cyan system with a compact sidebar.

## Applied settings

| Setting | Value |
| --- | --- |
| Theme | Clean |
| Primary color | `#2563EB` |
| Tint | Soft sky |
| Sidebar | Filled background with pill links |
| Search | Prominent |
| Corners | Rounded |

## Navigation pattern

Use page groups for workflows, not file-system folders. The sidebar should answer: "What am I trying to do next?"
