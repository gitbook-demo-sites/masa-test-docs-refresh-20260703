# Share links and access

Nimbus test docs are public for easy validation, but production docs should use the access model that matches the audience.

| Access model | Use when |
| --- | --- |
| Public | The docs are ready for anyone to read. |
| Share link | The docs are being reviewed with a limited group. |
| Visitor auth | Content should be restricted by identity or claims. |
| Internal workspace | Only teammates should collaborate on drafts. |

{% hint style="warning" %}
Before sending a share link, open it in a private browser window and confirm the visitor sees only the intended content.
{% endhint %}
