# Review process

Use change requests when the edit affects navigation, permissions, publishing, or visitor-facing guidance.

## Review checklist

| Check | Reviewer question |
| --- | --- |
| Accuracy | Is the workflow still true? |
| Clarity | Can a new teammate follow it without context? |
| Structure | Does the page belong in this sidebar group? |
| Search | Would Ask retrieve this page for the right question? |

## Approval rule

Small copy edits can publish after one review. Larger structural changes should be previewed on the live site before sharing broadly.
